How to use this dashboard:

  1. Copy the YAML above
  2. In Home Assistant, go to Settings - Dashboards
  3. Click Add Dashboard and create a new one
  4. Open the new dashboard, click the 3-dot menu - Edit Dashboard
  5. Click the 3-dot menu - Raw configuration editor
  6. Select all and paste the copied YAML
  7. Click Save
